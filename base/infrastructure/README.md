# Infrastructure Directory

Core infrastructure components organized by deployment wave.

## Purpose

Contains foundational infrastructure services deployed in the first three waves to establish the platform foundation.

## Structure

- `core/` - Wave 1: Essential services (Traefik, LocalStack)
- `operators/` - Wave 2: Kubernetes operators (CNPG, External Secrets)
- `config/` - Wave 3: Configuration and namespace setup
- `monitoring/` - Wave 4: Monitoring stack (Prometheus, Grafana)
